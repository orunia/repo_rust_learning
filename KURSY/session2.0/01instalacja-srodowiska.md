## CZESC 3: INSTRUKCJA INSTALACJI ŚRODOWISKA (WINDOWS)

## Wstęp
Ten dokument przeprowadzi Cię przez instalację narzędzi niezbędnych do tworzenia aplikacji w Rust i Flutter.

## Krok 1: Wstępne narzędzia
1.  **Git:** Pobierz i zainstaluj [git-scm.com](https://git-scm.com/).
2.  **Visual Studio Code (VS Code):** Pobierz i zainstaluj [code.visualstudio.com](https://code.visualstudio.com/).
3.  **C++ Build Tools (KRYTYCZNE dla Rusta):**
    * Wejdź na stronę Microsoft Visual Studio Downloads.
    * Pobierz "Build Tools for Visual Studio 2022".
    * Podczas instalacji zaznacz opcję: **"Desktop development with C++"** (Rozwój aplikacji klasycznych w języku C++). Bez tego Rust nie zadziała na Windows.

## Krok 2: Instalacja Rusta
1.  Wejdź na [rust-lang.org/tools/install](https://www.rust-lang.org/tools/install).
2.  Pobierz i uruchom `rustup-init.exe`.
